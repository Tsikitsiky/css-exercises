Boxes

A short essay on what the Box Model is

A Box Model is all you can see on a page. It is built by using CSS. There is two types of boxes in CSS, block boxes and inline boxes. A model box contain several parts: content box, padding box, border box and merging box wich all have its specific role.

A diagram of the box model

Description 1

Content-box
It is the area where the content like text, image or video of the document are displayed.
Padding-box
It is sited around the content as a white space area.
Border-box
It wraps the padding box area and the content box

Description 2

Border
It can be used as a shorthand property by putting all values together. It is better to use the long hand properties; border width, border style and border color to make it more specific.
Margin
In longhand peoperties, margin is margin top, margin bottom, margin left and margin right. It can be used a shorthand or detailed as a longhand properties.
Padding
Similar to margin, it is a shorthand for padding top, bottom, left and right; can be used as shorthand and longhand.
Display
It decides whether an element is displayed or not and if displayed, where it is displayed.
Box-sizing
It calculates the width and height of an element. It also can adjust the behaviour of a content-box and a border-box
