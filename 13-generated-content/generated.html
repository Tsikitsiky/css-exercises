Pseudo-elements

The special property that makes pseudo-elements work is the content property.

These functions can be used with that property's value:

  • a string,
  • an image,
  • nothing,
  • a counter

The various pseudo-elements are:

  • ::after (:after)
  • ::backdrop
  • ::before (:before)
  • ::cue
  • ::cue-region
  • ::first-letter (:first-letter)
  • ::first-line (:first-line)
  • ::grammar-error
  • ::marker
  • ::part()
  • ::placeholder
  • ::selection
  • ::slotted()
  • ::spelling-error

We use pseudo-elements to style a specific part of an element.

Counters

WE use counter fonction to count in CSS. These declaration are used if we want to use counter:

  • counter-reset = step;
  • counter-increment = steps;
  • content = steps.

    • To reset a counter we use the counter-reset property.

    Counters are useful for adjusting the appearance of a content in a document and can be used to number a list.

    Lists

    These properties are related to lists:

    • list-style
    • list-style-type
    • list-style-position
    • list-style-image

    There are three types of bullet style:

    • cirle,
    • square,
    • disk

    Yes, by using counter function we make an element to be styled as a list.
