Table, border-spacing, border-collapse, caption-side and empty-cells are related to table. These are their default values:

Properties Values
table-layout auto
border-spacing 2px
border-collapse separate
caption-side top
empty-cells show

Simulating HTML tables

let's take a look at these display values and the HTML elements which correspond them.

  • table = table
  • table-raw-group = tbody
  • table-header-group = thead
  • table-footer-group = tfoot
  • table-raw = tr
  • table-cell = td
  • table-column-group = colgroup
  • table-column = col
  • table-caption = caption
