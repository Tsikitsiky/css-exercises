Overflow

  • The default value is visible
  • The reason why we use this property is to deal with a content that doesn't fit its box.
  • The hight has to be set to make this property works effectively

Clip

  • The default value is auto
  • The use of it is to define which portion of an element is visible
  • Position has to be absolute or fixed to make it works

Clip-path

  • None is the default value
  • It is used to show or hide some parts of an element

Text-overflow: ellipsis

  • It put texts in ellipsis form (...)
  • To make it work, overflow and white-space properties must be set

In case of a content larger than its container, the browser use the overflow: visible property value as default way to deal with.

Ellipsis is a value of the text-overflow property which shows some parts of an element.It is different from ellipse because ellipse is a circle shape.

Text-overflow only affects content that is overflowing a block container element in its inline progression direction.
